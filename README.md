### 피부분석/설문조사 기반 화장품 추천 앱 개발 프로젝트
개발기간 : 2022.01.24 ~ 2022.02.08
<br>
링크 : <https://github.com/2021-SMHRD-KDT-AI-7/Debugging>
##### 1.  개요
- Faster R-CNN 객체 인식을 통해 여드름, 블랙헤드를 감지
- 바우만 테스트를 통한 피부 타입 분석
- 사용자에게 맞는 화장품을 추천해주고, 피부 분석 결과들을 기록한다.

##### 2.  스킬
- Java (Android Studio) 
- Python (Faster R-CNN 
- OracleDB
